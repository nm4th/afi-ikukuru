# 投稿スケジュール（1日5ツイート・INTJ特化）

## タイムテーブル

| 枠 | 時間 | タイミング | カテゴリ | 生成方法 |
|----|------|-----------|---------|---------|
| 朝 | 7:30 | 通勤電車 | 偏りあるある（笑える系） | `daily` で毎日生成 |
| 昼 | 12:15 | 昼休み | 奮闘記シリーズ（連載） | `series` で週次生成 |
| 夕 | 18:30 | 退勤中 | 偏りあるある（仕事系） | `daily` で毎日生成 |
| 夜① | 21:30 | 帰宅後 | 武器になった話＋誘導 | `daily` で毎日生成 |
| 夜② | 23:00 | 就寝前 | 気づき・内省 | `daily` で毎日生成 |

---

## 運用フロー

### 毎週日曜（週次）
```bash
# 来週の奮闘記シリーズを生成
python generator/generate.py series --theme "来週のテーマ"
```
→ 7話分をまとめて予約投稿に設定

### 毎日（日次）
```bash
# その日の4本を生成
python generator/generate.py daily
```
→ 朝・夕・夜①・夜② の4本を予約投稿に設定

### ネタ被り防止
- 生成するたびに `history/` に過去ツイートが蓄積される
- Claude APIが過去の内容を参照して、被らないツイートを生成する
- 長期運用しても同じようなネタにならない

週ごとにテーマを変えて連載する:

- Week 1: 「アプリを始めるまでの葛藤」
- Week 2: 「初マッチからデートまで」
- Week 3: 「初デートの準備と当日」
- Week 4: 「メッセージのやりとりで距離が縮まる話」
- Week 5: 「2回目のデートで手応えを感じる」
- Week 6: 「告白するかどうか悩む」
- Week 7: 「付き合ってからの日常」
- Week 8: 「喧嘩と仲直り（INTJの不器用さが出る）」
- Week 9: 「記念日のサプライズ計画」
- Week 10: 「偏ったままで幸せになれた振り返り」

## 枠ごとの狙い

### 7:30（通勤電車）
- 「朝からわかるw」でRT獲得
- 短め3〜5行のあるある
- 笑える系で1日のスタートを軽くする

### 12:15（昼休み）
- じっくり読む時間がある → 奮闘記シリーズの連載枠
- 月〜日で①〜⑦が順番に流れて物語になる
- 「続きが気になる」→ フォロー

### 18:30（退勤中）
- 仕事帰りの疲れ＋共感で拡散されやすい
- 仕事と恋愛の対比ネタが刺さる

### 21:30（帰宅後）★誘導枠
- 1日で一番反応が良い時間帯
- 「偏りが武器になった」転換ツイートを投稿
- 5〜10分後にリプライでPappy誘導
- 毎日誘導するが、切り口を全部変えて飽きさせない

### 23:00（就寝前）
- 一人の夜に刺さる感傷系
- 「わかる…」の深い共感でフォロワーとの絆を深める
- 誘導なし。純粋な共感コンテンツ
