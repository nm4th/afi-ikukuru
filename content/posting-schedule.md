# 投稿スケジュール（ランキング形式・インプレッション収益）

## タイムテーブル（1日5ツイート）

| 枠 | 時間 | 内容 |
|----|------|------|
| 朝 | 7:30 | ランキング（軽めテーマ） |
| 昼 | 12:15 | ランキング（メインテーマ・バズ狙い） |
| 夕 | 18:30 | ランキング（仕事帰りに刺さるテーマ） |
| 夜① | 21:30 | ランキング（夜の感情系テーマ・バズ狙い） |
| 夜② | 23:00 | ランキング or INTJのつぶやき |

※各ランキングは「5位→1位は↓」＋リプライで1位発表 = 1投稿あたり2ツイート

---

## 運用フロー

### 毎日
```bash
python generator/generate.py daily
```
→ 5テーマ分のランキング（本ツイート＋1位リプライ）を生成

### テーマ指定で1本だけ
```bash
python generator/generate.py single --theme "付き合ったら一途すぎる男のMBTI"
```

---

## バズを狙う時間帯

- **12:15 と 21:30 がメイン砲**（最もインプレッションが伸びる時間帯）
- 朝と夕方は安定枠（軽めのテーマでフォロワーとの接点を維持）
- 23:00は感傷系テーマ or INTJ個人のつぶやきで「人間くささ」を出す
