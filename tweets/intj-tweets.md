# サンプルツイート集（3形式）

参考用のサンプル。実運用ではジェネレーターで都度生成する。
形式A: 5→1位は↓ / 形式B: 1→5位 / 形式C: Tier表

---

## サンプル1: 付き合ったら一途すぎる男のMBTI

**本ツイート:**
```
付き合ったら一途すぎる男のMBTI

5位 ISFJ（擁護者）
→記念日を全部覚えてる。相手の好きなもの、嫌いなもの、全部メモしてる。

4位 ISTJ（管理者）
→浮気という概念が辞書にない。一度決めたら揺らがない。

3位 ENFJ（主人公）
→相手の幸せ＝自分の幸せ。尽くしすぎて自分を見失うレベル。

2位 INFJ（提唱者）
→「この人だ」と確信したら、もう他の人が見えなくなる。魂レベルで一途。

1位は↓
```

**リプライ:**
```
1位 INTJ（建築家）

好きになるまでに時間がかかる分、一度好きになったら揺らがない。

・そもそも人を好きになること自体がレア
・好きになった理由を論理的に説明できる＝感情だけじゃない
・将来設計に相手を組み込んでるので、簡単には手放さない

分析した僕が言うのもアレだけど、INTJの一途さは本物。
…自分で1位にするの恥ずかしいな。
```

## サンプル2: 冷めたはずなのにまた好きになる男のMBTI

**本ツイート:**
```
冷めたはずなのにまた好きになる男のMBTI

5位 ESTP（起業家）
→冷めたと思っても、久しぶりに会うと楽しさが戻ってきてしまう。

4位 ENFP（運動家）
→熱が冷めても、相手の新しい一面を見た瞬間に再燃する。

3位 ENTP（討論者）
→冷めたと判断しても、面白い話をされると全部リセットされる。

2位 ISFP（冒険家）
→距離を置いても、相手の感性に触れた瞬間にまた引き込まれる。

1位は↓
```

**リプライ:**
```
1位 INFP（仲介者）

一度好きになった人への気持ちが完全に消えることがない。

・冷めたと思っても、ふとした瞬間に思い出す
・相手の良いところを美化して記憶に残してしまう
・「もしあの時こうしていたら」のifを永遠に考え続ける

INFPの恋心、アンインストールできない仕様になってる。
```

## サンプル3: 好きな人の前で挙動不審になるMBTI

**本ツイート:**
```
好きな人の前で挙動不審になるMBTI

5位 ISFP（冒険家）
→急に黙る。普段は穏やかなのに、好きな人の前だけ石になる。

4位 INTP（論理学者）
→早口で趣味の話を始めてしまい、後で死ぬほど後悔する。

3位 INFP（仲介者）
→目を合わせられない。話しかけられると声が裏返る。

2位 INFJ（提唱者）
→相手のことを考えすぎて、何を話せばいいかフリーズする。

1位は↓
```

**リプライ:**
```
1位 INTJ（建築家）

普段は冷静沈着なのに、好きな人の前だけバグる。

・会話をシミュレーションしてきたのに全部飛ぶ
・平常心を装おうとして逆に不自然になる
・帰宅後に「あそこでああ言えばよかった」の反省会が3時間

仕事のプレゼンは完璧にこなすのに。
…これ僕のことです。
```

## サンプル4: 形式B（1→5位・1ツイート完結）

```
【MBTIランキング｜サプライズが得意なタイプトップ5】

1位 ENFJ：相手が何を喜ぶかを完璧に把握した上でサプライズを仕掛ける。外れることがほぼない

2位 ESFJ：準備に全力を注ぐ。プレゼント・演出・タイミング、全部考えてある

3位 ENFP：予想外のアイデアが湧き出る。「そんなことしてくれるとは」という驚きを作るのが得意

4位 ESTP：当日のノリと行動力でサプライズを仕掛ける。準備は少ないが決まると最高

5位 ESFP：テンションとノリで相手を驚かせる。雰囲気づくりが天才的
```

## サンプル5: 形式C（Tier表）

```
【恋愛MBTI｜嫉妬深さTier表】

【S】嫉妬の権化

INFJ（静かな嫉妬）
→表には出さないが、内心では燃えている。気づいた時には手遅れなレベルで溜め込む。

ISFJ（献身ゆえの嫉妬）
→「私がこんなに尽くしてるのに」が爆発する。普段優しい分、ギャップが怖い。

【A】かなり嫉妬する

ENFJ（守りたい嫉妬）
→相手を大切にしすぎて、近づく人間全員が脅威に見える。

ESFJ（確認したい嫉妬）
→「私のこと好き？」の確認が増える。不安を言葉にできる分まだ健全。

INTJ（分析する嫉妬）
→嫉妬してる自分を分析し始める。感情と論理の間で混乱する。

【B】状況次第

ENFP・INFP・ISFP・ESTP
→信頼してる間は平気。でも裏切りを感じた瞬間にスイッチが入る。

【C】あまり嫉妬しない

ENTP・INTP・ISTP・ESTJ
→そもそも束縛が嫌い。嫉妬するくらいなら関係を見直す合理派。

【D】嫉妬の概念が薄い

ENTJ・ESFP
→自分に自信があるので嫉妬しにくい。「嫉妬する暇があったら自分を磨く」タイプ。
```
